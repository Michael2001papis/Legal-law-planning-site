# עולם הסרטים של MP – סיכום פרויקט

## סקירה כללית

פלטפורמת סרטים בסגנון Netflix לצורכי לימוד והדגמה. פרויקט Full-Stack עם React, Node.js, MongoDB.

---

## מה תוקן ועודכן (סיום)

### 1. תיקון 404 ברענון
- **בעיה:** רענון בדפים כמו `/home` או `/movie/123` החזיר 404
- **פתרון:** הוספת `rewrites` ב־`vercel.json` כך שכל הנתיבים (חוץ מ־`/api`) מפנים ל־`index.html` – React Router מטפל בניווט

### 2. התחברות ללא שרת (Vercel)
- **בעיה:** אין Backend ב־Vercel – אי אפשר להירשם או להתחבר
- **פתרון:** כפתור **"התנסה במצב דמו"** בעמודי ההתחברות וההרשמה
- לחיצה על הכפתור נכנסת ישירות לאתר כמשתמש דמו (מנוי Diamond)
- אפשר לעיין בממשק בלי להתקין שרת

### 3. ביצועים
- בנייה עם Vite (ES2020)
- Code splitting – `react`, `react-dom`, `react-router-dom` בחבילה נפרדת
- minify ל־JS ו־CSS
- `loading="lazy"` לתמונות

### 4. תמיכה בדפדפנים
- Target: ES2020 – תאימות לדפדפנים מודרניים
- `-webkit-backdrop-filter` לתמיכה ב־Safari
- `touch-action: manipulation` להפחתת עיכוב טאצ׳ים במובייל

### 5. מובייל ורספונסיביות
- **1024px ומטה:** ניווט מקוצר, מרווחים מצומצמים
- **768px ומטה:** תפריט מינימלי, Hero קטן יותר, כרטיסים קטנים יותר
- **480px ומטה:** התאמות ל־mobile
- **prefers-reduced-motion:** צמצום אנימציות למשתמשים שביקשו זאת

## מבנה הפרויקט

```
├── client/                 # React + Vite
│   ├── src/
│   │   ├── components/     # Layout, MovieRow
│   │   ├── pages/          # כל המסכים
│   │   ├── services/       # API, planPolicy
│   │   ├── store/          # authStore (Zustand)
│   │   └── styles/
│   └── index.html
├── server/                 # Node.js + Express
│   └── src/
│       ├── config/         # planPolicy
│       ├── middleware/     # auth, errorHandler
│       ├── models/         # User, Movie, Session, etc.
│       └── routes/         # auth, movies, user, admin
├── vercel.json             # Deploy + SPA rewrites
├── package.json
└── .env.example
```

## הרצה מקומית (עם Backend)

```bash
# התקנה
npm install
cd client && npm install
cd ../server && npm install

# MongoDB חייב לרוץ
cd ../server && npm run seed    # נתוני דוגמה + admin

# הפעלה
cd ..
npm run dev
```

**משתמש מנהל:** admin@mp.com / admin123

## הרצה ב־Vercel (Frontend בלבד)

- רק ה־client עולה
- אין MongoDB ולא Backend
- שימוש ב־**"התנסה במצב דמו"** כדי לגלוש באתר

---

## עיצוב (Netflix-style)

- **Hero** – מסך פתיחה גדול עם רקע
- **שורות אופקיות** – גלילה צדית לסרטים
- **כרטיסי סרטים** – הגדלה בצלילה, overlay עם פרטים
- **Navbar** – שקוף בהתחלה, זכוכית בגלילה
- **דף סרט** – Hero גדול, מטא, כפתורי פעולה
